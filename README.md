# webWacode

Landing-only hackathon website with a Liquid Glass design language:
- Blur + translucent glass surfaces
- Subtle 1px white edge highlights
- Large smooth radii and soft floating shadows
- Spring-like motion on hover/press

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Demo content configuration

Update links in `src/content.js`:
- `videoEmbedUrl`
- `videoOpenUrl`
- `pitchDeckEmbedUrl`
- `pitchDeckOpenUrl`
